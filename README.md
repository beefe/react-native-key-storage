# react-native-key-storage
the android nativeModule key-storage for react-native 

## Installation and How to use

#### Step 1 - RNPM Install
```shell
$ npm install rnpm -g
```

#### Step 2 - NPM Install
```shell
$ npm install --save react-native-key-storage
```

#### Step 3 - Add Link
```shell
$ rnpm link react-native-key-storage
```
